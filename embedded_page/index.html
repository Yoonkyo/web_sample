수업목표

임베디드 시스템 이해하기

임베디드 시스템은 컴퓨터 하드웨어와 소프트웨어가 조합되어 특정한 목적을 수행하는 시스템입니다. 수업을 통해 임베디드 어떤 구조와 기능을 갖는지, 그 사례에는 무엇이 있는지 알아볼 것입니다.

라즈베이파이로 IoT 장비 개발하기

작은 컴퓨터라고 불리는 라즈베리파이를 사용하여 직접 임베디드 시스템을 개발해볼 것입니다. 기본 센서의 동작 방법을 배우고, 여러분들이 직접 실생활에 유용한 장비를 개발할 것입니다.

전공자로서 알아야 할 기본지식 확인

사회에서 Computer Science 전공자라고 얘기하기 위해 어느 정도 통용되는 지식이 있습니다. 이러한 지식 습득은 물론, 임관 후에도 직접 찾아볼 수 있는 자료 등을 공유할 예정입니다.

관심있는 연구 분야 선정하기

조만간 여러분은 졸업 논문을 작성하게 됩니다. 좀 더 의미있는 논문 작성을 위해 담당 교수와 면담 후 관심 분야를 탐색하고, 이를 통해 연구 주제도 찾아볼 것입니다.
